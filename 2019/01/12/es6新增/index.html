es6新增

Posted on 2019-01-12 | In js , 笔记
Words count in article: 2.1k 字 | Reading time ≈ 10 分钟
记录一下 JavaScript 新增的知识点

新增类型

进制

0b 二进制 0八进制 0x十六进制

symbol

基本类型，返回值唯一

使用：属性私有化，只读

字符串拓展

  • Unicode 表示法 {}完整表示

  • (http://www.unicode.org/) 表情 (http://www.unicode.org/emoji/charts)

  • 模板字符串 保持字符串格式 （空格和缩进）

  • 表达式解析 ${} 可以使用在属性里面

Map

对象的键值只能是字符串；如果不是字符串，会尝试调用对象的 toString 方法

Map 允许属于各种类型（包括对象）；key 按照添加顺序排列

可以使用在私有数据保护

创建：

属性：size

方法：set(key,value)、get、delete、has、clear、keys、values、entries、forEach

  • WeakMap 键值必须是对象，弱引用，GC机制不考虑回收问题（new Person 是强引用，两处保存引用）。多用于上述数据保护的垃圾回收

Set

集合，唯一值不会重复

Set 函数 传入可迭代对象

属性：size

方法： add、has、delete、clear、keys、values、entries、forEach

let、const

新增块级作用域 {}（包括使用 if 和 for 里面的）

  • var ：支持预解析、可以重复声明、不支持作用域
  • let ：不支持预解析、不允许重复声明、支持块级
  • const：不支持预解析、不允许重复声明、支持块级、必须初始化
    • 数组、对象可以添加值
    • 不允许修改数组：const arr = Object.freeze([1,2,3]) ，但是嵌套数组允许修改

对象拓展

  • 间接表示 键值和属性值一样时，简写

  • 属性名表达式 键值接收表达式

解构赋值

从数组和对象提取值，对变量进行赋值；左侧表示变量，不表示数组的意思

  • 数组

  • 对象

  • 多重解构

拓展运算符

…把数组/对象转成参数序列，可用于 数组传参、数组合并

  • [‘a’,‘b’,‘c’] => ‘a’,‘b’,‘c’
  • { left:100,top:200 } => left:100,top:200

迭代

  • 实现对象迭代

函数拓展

  • 设置默认参数 必填在前，可选在后

  • rest剩余参数… 赋值给紧邻变量（数组）

  • 箭头函数 函数声明 => 函数体 () => {} ；

    参数只有一个，省略()；一条语句，可以省略{return}

    • this 创建时就指向创建期上下文 ；即使赋值也是原有指向
      • setTimeout原本指向 window，使用箭头函数指向当前
    • 没有 arguments
    • 不能作为生成器函数

内置对象

  • Set、WeakSet
  • Map、WeakMap

函数拓展

String

  • startWith() 以…开始 ；返回布尔值
  • endwith()
  • includes()
  • repeat() 重复次数

Number

  • isFinite()、isNaN()、parseInt、parseFloat

Array

  • includes(value)
  • forEach( function ( item,index,array ) { },content) // 第二个参数传入 this 指向
  • every 都真为真 实现全选
  • some 有真为真 实现多选一
  • filter 返回真的值
  • map 映射
  • reduce(function(prev,current){}, 0) // 0 表示初始值
  • Array.from() 转换成数组
  • fill(str,start,end) 将所有元素替换

Obj

  • assign(obj1,obj2) 浅拷贝
  • create(obj) 赋值给的 prototype
  • freeze
  • definePrototype(obj,‘key’,{ }) // 定义属性、描述符（默认不可枚举、删除）

Class 类

原有方法：

新的方法：

  • constructor 方法是构造方法，this 代表实例对象，即ES5 的Miaov 函数构造方法，对应ES6的Miaov类的构造方法
  • constructor必须有；不显式调用仍会默认使用
  • print 方法，不需要function关键词和,逗号
  • 类的所有方法，仍然在类的 prototype 上
  • 类中的方法不可枚举

继承

  • ES5 调用父类 parent.call(this,value) ; ES 6 使用 super
  • 创建时 constructor里面没有this ；之外不能使用 super
  • 如果子类有自己的构造函数，必须调用父类**

异步 Promise

  • 对象 new Promise(cb) ===> 基本使用 Pending 进行中 Resolved 已完成 Rejected 已失败

  • 原型方法：

    • then 成功调用 ；then 调用无论成功失败，默认会返回成功的Promise实例

    • catch 失败调用 ；只有前面的 then 没有第二个参数，才能触发；也会返回成功的Promise

  • 静态方法

    • all 将多个 Promise 实例包装成新的实例
      • 当所有实例的状态都为 resolved，all 的实例状态才为 resolved；返回值组成一个数组，传给 then 中的 resolve 函数
      • 只要有一个被 rejected，all 的状态就是 rejected ；第一个 rejected 的返回值传给回调函数
    • Promise.race([p1,p2]) // 类似 some ，先跑完先执行
    • resolve

Generator

function* fn()，内部使用 yield ，定义遍历器的成员

只能使用 next(value) 调用返回的迭代函数，传入参数

实现异步

async await
