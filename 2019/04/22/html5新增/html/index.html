DOM

节点

节点类型

  • 元素节点。DOM的原子是元素节点，如 ul p div之类。这些元素布局形成网页结构。标签名就是元素名。
  • 文本节点。文本内容。
  • 属性节点。对元素做出具体描述，如 title
节点属性
  • childNodes 所有子元素数组（不只是元素节点）
  • parentNode
  • nodeType 节点类型，1 元素 2属性 3 文本等等
  • nodeValue节点值，元素节点值为 null
  • firstChild lastChild
  • nextSibling

节点操作

节点获取
  • document.getElementById(id) 返回单个节点对象
  • element.getElementByTagName(tag) 返回节点数组
  • element.getElementByClassName(class) 返回节点数组，可以传入多个类名，空格隔开
  • document.querySelectorAll()
属性操作
  • el.getAttribute(attr) 获取属性
  • el.getAttribute(attr,value) 设置属性
创建标记

HTML 方法：document.write 和 innerHTML

DOM 方法：

document.createElement(nodeName)创建元素节点

document.createTextNode(text)创建文本节点
